📊 预定利率研究值计算逻辑与数据源说明

基于公开数据源的预定利率研究值计算模型解析与回溯验证报告

报告日期：2026年7月 | 计算模型版本：v2.1（动态税收溢价 + 分段累加法）

📋 报告摘要

核心结论

本报告基于完全公开的数据源重构了预定利率研究值的计算模型，并对2024Q4至2026Q1共6个季度进行了回溯验证。结果显示（采用分段累加系数实现 + 动态税收溢价）：

  • 所有季度的预测值与实际值差异均在 5 BP 以内，达到"高度吻合"标准
  • 平均绝对误差：1.42 BP
  • 最大误差：2.69 BP (2025Q1)
  • 最小误差：0.42 BP (2024Q4)

这表明：使用公开数据（LPR、六大行定存利率、10年期国债收益率）并配合动态税收溢价（国开债/进出口债与国债收益率之差的250/750日移动平均），可较准确地还原预定利率研究值的计算过程。

🔢 预定利率研究值计算公式

第一步：计算参考利率

参考利率 = 6个月移动平均[ (5年期LPR + 5年期定期存款利率) ÷ 2 ] 其中： - 5年期LPR：取中国货币网公布的5年期以上LPR - 5年期定期存款利率：取六大国有银行（工农中建交邮）挂牌利率的算术平均值 - 6个月移动平均：取计算日前6个月的月度数据计算平均值

第二步：计算税收溢价

税收溢价 = MAX( 国开债收益率 - 国债收益率, 进出口债收益率 - 国债收益率 ) 的移动平均 其中： - 使用250日移动平均（对应参考利率的6个月窗口） - 使用750日移动平均（对应基础回报水平的长期趋势） - 取国开债和进出口债两者与国债利差的最大值

✅ 数据说明（已动态获取）

税收溢价所需的国开债/进出口债日频收益率已动态获取，不再使用静态近似。混合数据源如下：

  • 2026-04-01 至 2026-07-08（近67天）：中国货币网真实值（国开 CYCC021 / 进出口 CYCC024）
  • 2002-01-04 至 2026-03-31（其余6055天）：中债估值 Excel 补齐
  • 合计 6122 条，cdb/ieb 零缺口

税收溢价按 250/750 日移动平均实时计算，取国开、进出口与国债利差两者最大值。以2025Q4为例：spread_250 = 0.1386%、spread_750 = 0.1643%。

第三步：计算基础回报水平

基础回报水平 = MIN( 10年期国债收益率250日移动平均 + 税收溢价(250MA), 10年期国债收益率750日移动平均 + 税收溢价(750MA) ) 说明： - 取短端（250日）和长端（750日）两者中的较低值，体现审慎原则 - 10年期国债收益率数据来自中国货币网日频数据

第四步：计算系数调节前预定利率

系数调节前利率 = MIN( 参考利率, 基础回报水平 )

第五步：分段系数调节

最终预定利率 = Σ [ 每段区间长度 × 对应系数 ] 分段调节系数表： ┌─────────────────┬──────────┐ │ 利率区间 (%) │ 调节系数 │ ├─────────────────┼──────────┤ │ (0.00, 1.00] │ 1.00 │ │ (1.00, 2.00] │ 1.00 │ │ (2.00, 2.50] │ 0.95 │ │ (2.50, 3.00] │ 0.95 │ │ (3.00, 3.50] │ 0.50 │ │ (3.50, 4.00] │ 0.50 │ │ (4.00, 10.00] │ 0.30 │ └─────────────────┴──────────┘ 示例： 若系数调节前利率 = 2.3534% 计算过程： (1.00 - 0.00) × 1.00 = 1.00% (2.00 - 1.00) × 1.00 = 1.00% (2.3534 - 2.00) × 0.95 = 0.3357% 最终 = 1.00 + 1.00 + 0.3357 = 2.3357% ≈ 2.34%

💡 结论与建议

主要结论

  1. 模型有效性：使用公开数据（LPR、六大行定存利率、10年期国债收益率）配合动态税收溢价，可以较准确地还原预定利率研究值的计算过程。回溯验证显示6个季度误差均在5BP以内（平均1.42BP）。
  2. 数据完整性：税收溢价所需的国开债/进出口债日频数据已通过混合数据源（近67天中国货币网真实值 + 6055天中债估值 Excel 补齐，共6122条零缺口）获取，模型完整性显著提升。
  3. 计算逻辑：预定利率研究值的计算逻辑清晰，主要包括参考利率、基础回报水平、分段系数调节三个核心步骤。

改进建议

  1. 数据源增强：持续通过中国货币网增量积累国开/进出口债真实日频值，进一步缩短对中债估值 Excel 的依赖，保持税收溢价时效。
  2. 实时更新：建立自动化的数据更新机制，每次LPR或存款利率调整后自动重新计算预测值。
  3. 敏感性分析：分析各输入变量（LPR、定存利率、国债收益率）对最终研究值的影响程度，识别关键驱动因子。
  4. 预测功能：基于当前数据，建立2026年各季度的预测模型，为业务决策提供参考。
