第一章：计算公式解析

根据《预定利率研究值-使用版.xlsx》文件分析，预定利率研究值的计算基于以下逻辑：

步骤1：计算基础回报水平
基础回报水平 = Min(MA250 + 税收溢价₁, MA750 + 税收溢价₂)
步骤2：确定参考利率
参考利率 = 2.70% （固定值）
步骤3：计算差值
差值 = 参考利率 - 基础回报水平
步骤4：查表确定系数调整
根据差值所在区间，查表确定调整系数
最终预定利率 = 基础回报水平 - 系数调整

💡 关键参数说明

  • MA250：10年期国债到期收益率的250日移动平均
  • MA750：10年期国债到期收益率的750日移动平均
  • 税收溢价₁：约0.14%（用于补偿国债利息免税的税收差额）
  • 税收溢价₂：约0.22%（750日移动的税收溢价更高）

4. 参考利率
2.70% （固定）
5. 差值
2.70% - 2.3532% = 0.3468%
6. 系数调整

附录：Python公式实现

以下是预定利率研究值计算的Python实现代码： 

import pandas as pd
import numpy as np

def calculate_reserve_rate(df_bond, tax_premium_250=0.001386, tax_premium_750=0.002224):
    """
    计算预定利率研究值

    参数:
    - df_bond: DataFrame, 包含'日期'和'10Y收益率'列
    - tax_premium_250: float, MA250的税收溢价（默认0.1386%）
    - tax_premium_750: float, MA750的税收溢价（默认0.2224%）

    返回:
    - DataFrame with MA250, MA750, 基础回报水平, 预定利率
    """

    # 计算移动平均
    df_bond['MA250'] = df_bond['10Y收益率'].rolling(window=250, min_periods=250).mean()
    df_bond['MA750'] = df_bond['10Y收益率'].rolling(window=750, min_periods=750).mean()

    # 计算基础回报水平
    df_bond['基础回报水平'] = df_bond.apply(
        lambda row: min(row['MA250'] + tax_premium_250, 
                       row['MA750'] + tax_premium_750)
        if pd.notna(row['MA250']) and pd.notna(row['MA750']) else np.nan,
        axis=1
    )

    # 参考利率
    REFERENCE_RATE = 0.027  # 2.7%

    # 计算系数调整
    def get_adjustment(diff):
        """根据差值查表获取系数调整"""
        if diff <= 0.01:      # [0, 1%)
            return 0.01
        elif diff <= 0.02:    # [1%, 2%)
            return 0.01
        elif diff <= 0.025:   # [2%, 2.5%)
            return 0.003355
        elif diff <= 0.03:    # [2.5%, 3%)
            return 0
        elif diff <= 0.035:   # [3%, 3.5%)
            return 0
        elif diff <= 0.04:    # [3.5%, 4%)
            return 0
        else:                  # [4%, +∞)
            return 0

    # 应用调整
    df_bond['差值'] = REFERENCE_RATE - df_bond['基础回报水平']
    df_bond['系数调整'] = df_bond['差值'].apply(get_adjustment)
    df_bond['预定利率'] = df_bond['基础回报水平'] - df_bond['系数调整']

    return df_bond
